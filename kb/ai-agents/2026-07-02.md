## arXiv (arxiv-rss/cs.MA): M2Note: Continual Evolution of Vision Language Models via Mistake Notebook Learning

- M2Note enables continual evolution of VLMs via a "mistake notebook" that records and replays failures.
- Avoids full SFT/RL cycles by targeting only recurring error patterns (skipped checks, misapplied rules, hallucinations).
- Updates model behavior incrementally without catastrophic forgetting of prior capabilities.
- Reports gains on multimodal reasoning benchmarks vs. vanilla continual-fine-tuning baselines.

**Why it matters:** Cheaper continual-learning path for VLMs — relevant to anyone maintaining multi-modal agents that drift over time.

**Tags:** #continual-learning #vlm #multimodal #memory #fine-tuning

**Source URL:** https://arxiv.org/abs/2607.00685

## arXiv (arxiv-api): AutoMem: Automated Learning of Memory as a Cognitive Skill

- Treats memory management as a trainable metacognitive skill for LLM agents (AutoMem).
- Promotes file-system operations to first-class actions so the model decides what to encode/retrieve/organize.
- Meta-learns memory policies rather than hard-coding retrieve-then-read patterns.
- Improves long-horizon task performance over fixed RAG/memory-tool baselines.

**Why it matters:** Reframes memory as a learned skill, not plumbing — key direction for agents that need to operate over weeks of context.

**Tags:** #memory #metacognition #llm-agents #long-horizon #tool-use

**Source URL:** http://arxiv.org/abs/2607.01224v1

## arXiv (arxiv-api): ASPIRE: Agentic /Skills Discovery for Robotics

- ASPIRE: continual-learning system that autonomously writes and refines robot control programs.
- Uses code-as-policy — the agent generates Python policies that are executed and tested in simulation.
- Iterates on failures to discover new skills without manual programming.
- Targets long-horizon manipulation tasks where pre-defined skill libraries fall short.

**Why it matters:** Robotics analog of AutoMem — closed-loop skill discovery for embodied agents, complementary to LLM-tool agents.

**Tags:** #robotics #code-as-policy #continual-learning #embodied-ai #agentic

**Source URL:** http://arxiv.org/abs/2607.00272v1
